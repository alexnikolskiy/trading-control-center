# Runbook: Release Platform Capability

## Steps

1. Confirm platform behavior and public contract.
2. Update `trading-platform`.
3. Update `trading-platform-sdk` if consumers are affected.
4. Update mock platform and contract fixtures.
5. Update affected consumers.
6. Run validation gates.
7. Confirm Phoenix traces, logs, metrics, and alerts.
8. Deploy in the agreed environment order.
9. Record rollout notes.
