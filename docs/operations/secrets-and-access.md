# Secrets And Access

## Rules

- Do not commit secrets to this repository.
- Keep access procedures in source-of-truth infrastructure documentation.
- Record ownership and escalation paths here at the coordination level.
- Use least-privilege access for agents and humans.

## Access Areas

| Area | Expected owner |
| --- | --- |
| Production platform | Platform owner |
| SDK publishing | SDK owner |
| Observability dashboards | Operations owner |
| Agent workflow credentials | Lab owner |
| Operator UI access | Office owner |
| Backtest data | Backtester owner |
